# Linux

## 常见命令

cd:进入目录

cdmir:创建文件夹

rm:删除文件

find:查找文件

cat:打开文件

top:查看CPU使用情况

ps:查看进程

ls:查看目录

cp:复制文件

mv:移动文件

### 面试问题

#### linux打开文件是一次性全部加载吗

不是，内存有物理内存和虚拟内存，虚拟内存在硬盘上，会根据情况加载到内存中。

#### linux怎么连续查看文件

cat a.txt>>b.txt(把a的内容放到b后面，不会覆盖)

#### 查看文件最后几行

cat filename|tail -n 1000(从1000行开始显示就是+1000，-f是监视文件尾部内容)
